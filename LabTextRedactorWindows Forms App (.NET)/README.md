# Windows Forms Text Editor

Simple text editor application built with C# and Windows Forms, extended with a WPF font settings window.

## Features

- Create new file
- Open existing .txt files
- Save changes
- Exit application
- Status bar with line and character counter
- Keyboard shortcuts (Ctrl+N, Ctrl+O, Ctrl+S)
- **Font Settings (WPF)** — change font, size, bold and italic with live preview

## Technologies

- C#
- .NET Framework 4.7.2
- Windows Forms
- Windows Presentation Foundation (WPF)
- System.IO

## How to Run

1. Open project in Visual Studio
2. Build the solution
3. Press F5 to start

## Project Structure

- `Form1.cs` – main UI logic
- `Form1.Designer.cs` – UI layout and menu definition
- `FontSettingsWindow.xaml` – WPF font settings window layout
- `FontSettingsWindow.xaml.cs` – WPF font settings window logic
- `Program.cs` – application entry point
- `App.config` – configuration file
